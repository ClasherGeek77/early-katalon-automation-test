# UNI Early Katalon Automation Test 🧪
A legacy automation project built using **Katalon Studio**. This repository serves as an archive for early test cases, object repositories, and custom keywords.

## Project Structure
- `Test Cases/`: Automated test scenarios.
- `Object Repository/`: UI element mappings.
- `Scripts/`: Underlying Groovy scripts for test cases.
- `Test Suites/`: Orchestrated groups of test cases.

## Tools Used
- Katalon Studio
- Groovy
